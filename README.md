# HexaSwift Mobile App

> Flutter app for remotely controlling a hexapod robot over HTTP with device pairing, real-time navigation, and choreographed dance routines. VLC/WebSocket livestreaming shows the robot camera feed; Express/MongoDB backend handles JWT auth and command routing.
